# 🤖 AI Tester Agent — Mission Srijan 26
## Srijan 2026 by Atos Global | AI-Agent Driven, Context-Aware Release Testing

### Tech Stack:
- Python 3.10+ | Streamlit | Gemini AI | LangChain | ChromaDB RAG | Playwright | Jira REST API

### Setup:
1. Install: `pip install -r requirements.txt && playwright install chromium`
2. Add GROQ_API_KEY in .env file
3. Run: `streamlit run app.py`
4. Open: http://localhost:8501
